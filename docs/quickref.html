Running murphy (the web ui)

C:\git-projects\murphy_sdk\web_workbench>workbench.py
And also:
C:\git-projects\murphy_sdk\model_extraction_helpers>python -m SimpleHTTPServer 8901
Then open a browser and go to http://127.0.0.1:8090/

Extracting a model

set PYTHONPATH=C:\git-projects\murphy_sdk
Be sure the local files are served:
C:\git-projects\murphy_sdk\model_extraction_helpers>python -m SimpleHTTPServer 8901
Then do:
C:\git-projects\murphy_sdk\7zip_modeller>extract_7zip_win_scraper.py

Adding a model to the web workbench

Edit \git-projects\murphy_sdk\web_workbench\config.json and add it there, for example add this at the end right before the ]:
, {"file": "../7zip_modeller/7zipWinScraper/7zipWinScraper.json"}
(be sure the opening [ is matched with one ] at the end!)

Creating / running a manually written test script

Take a look at 7zip_modeller/test_scripts/test_7zip_installation.py for a sample (and simple) script
To execute it ensure that local files are served and run test_7zip_installation.py
